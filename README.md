# AK-47 セレクター＆発射シミュレーター

AK-47アサルトライフルの特徴的な「セレクターレバー（安全装置・射撃モード切り替え）」の挙動と、発射メカニズムを視覚的に体験できるHTML5 Canvasシミュレーターです。

一般的な現代の銃とは異なり、安全装置からレバーを下げた際、**「セミオート（単射）」ではなく「フルオート（連射）」が最初に噛み合う**というAK-47特有の実戦的レイアウトを忠実に再現しています。

## 🚀 特徴

- **リアルなセレクター配置:** 
  上から順に `SAFE（安全）` ➔ `AUTO（連射）` ➔ `SEMI（単射）` というAK-47特有の配置。
- **インタラクティブ操作:** 
  画面上のセレクターレバーを直接クリックして切り替えることができます。
- **2つの射撃モード:**
  - **AUTO (連射):** トリガーボタンを押し続けている間、約600発/分のサイクル（100ms間隔）で弾丸を連射します。
  - **SEMI (単射):** ボタンを押し続けても1発しか発射されず、再度撃つにはトリガーを弾き直す必要があります。
- **視覚効果:** 発射に合わせて変化するマズルブラスト（銃口の炎）と残弾数カウンターを搭載。

🛠 使い方
1. ブラウザで動かす（簡単）
index.html を Google Chrome、Edge、Safari などのブラウザにドラッグ＆ドロップするだけで、すぐにビジュアル版シミュレーターが起動します。

2. TypeScript版（ロジック）をターミナルで動かす
CUI環境（ログ出力のみ）で内部ロジックの動作を確認したい場合は、以下の手順で実行できます。

Bash
# 必要ライブラリのインストール
npm install

# TypeScriptコードの実行
npx ts-node index.ts
💡 AK-47の設計雑学
実物のAK-47でセレクターを「安全」位置からパニック状態で勢いよく一番下まで引き下げると、レバーは必ず一番下の「SEMI（単射）」で行き止まります。
これは、戦場で恐怖に陥った兵士がトリガーをハダカで引きっぱなしにして弾薬を無駄遣いするのを防ぎ、強制的に落ち着いて狙わせる（単射にする）ための、天才的な実戦設計と言われています。本シミュレーターでもその切り替え順序を体感できます。
